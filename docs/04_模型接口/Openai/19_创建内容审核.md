# 创建内容审核

对文本是否违反 OpenAI 的内容政策进行分类

## 请求参数
Header 参数生成代码Authorizationstring 必需示例:sk-Body 参数application/json生成代码inputstring 要分类的输入文本必需modelstring 必需有两种内容审核模型可用：text-moderation-stable和text-moderation-latest。 默认值text-moderation-latest将随着时间的推移自动升级。这可确保您始终使用我们最准确的模型。如果您使用text-moderation-stable，我们将在更新模型之前提供提前通知。的准确度text-moderation-stable可能略低于 的准确度text-moderation-latest。

示例
## 返回响应
🟢200成功application/json生成代码Body生成代码idstring 必需审核任务的唯一标识符

modelstring 用于审核的模型必需resultsarray [object] 审核结果的详细信息必需categoriesstring 必需审核模型检测到的所有分类标签

category_scoresstring 必需审核模型对所有分类标签的得分

flaggedstring 必需审核模型是否将输入内容标记为可疑内容

请求示例请求示例ShellJavaScriptJavacURLcURL-WindowsHttpiewgetPowerShell
```
curl --location --request POST 'https://4sapi.com/v1/moderations' \
--header 'Authorization: sk-' \
--header 'Content-Type: application/json' \
--data-raw '{
    "model": "omni-moderation-latest",
    "input": [
        {
            "type": "text",
            "text": "...text to classify goes here..."
        },
        {
            "type": "image_url",
            "image_url": {
                "url": "https://github.com/dianping/cat/raw/master/cat-home/src/main/webapp/images/logo/cat_logo03.png"
            }
        }
    ]
}'
```

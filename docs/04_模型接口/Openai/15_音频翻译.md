# 音频翻译


## 请求参数


### Header 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `Accept` | string | 必需 | 示例: application/json |
| `Authorization` | string | 必需 | 示例: |
| `sk-` |  |  |  |
| `Content-Type` | string | 必需 | 示例: multipart/form-data |


### Body 参数 (multipart/form-data)

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `file` | file | 可选 | 要翻译的音频文件，支持以下格式：mp3，mp4，mpeg，mpga，m4a，wav 或 webm。 |
| `model` | string | 可选 | (要使用的模型的 ID，目前仅有 whisper-1 可用。 |
| `prompt` | string | 可选 | 一个可选的文本，用于指导模型的风格或继续先前的音频片段。 prompt 应为英文。 |
| `response_format` | string | 可选 | 转录输出的格式，支持以下选项之一：json、text、srt、verbose_json 或 vtt。 |
| `temperature` | number | 可选 | 采样温度，介于 0 和 1 之间。 较高的值（如 0.8）将使输出更随机，而较低的值（如 0.2）将使其更加专注和确定性。 如果设置为 0，则模型将使用对数概率自动增加温度，直到达到一定的阈值。 |

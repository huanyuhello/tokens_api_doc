# 生成歌曲


## 请求参数


### Header 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `Accept` | string | 必需 | 示例: application/json |
| `Authorization` | string | 必需 | 示例: |
| `sk-` |  |  |  |
| `Content-Type` | string | 必需 | 示例: application/json |


### Body 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `必填` |  |  |  |
| `prompt` | string | 必需 | 歌词内容,在自定义模式下需提供 灵感模式无需,自定义模式必需。 |
| `tags` | string | 必需 | 歌曲风格标签,使用逗号分隔,在自定义模式下需提供 灵感模式无需,自定义模式必需 |
| `mv` | string | 可选 | 模型版本,可选值:chirp-v3-0、chirp-v3-5,默认为 chirp-v3-0 |
| `title` | string | 必需 | 歌曲标题,在自定义模式下需提供 灵感模式无需,自定义模式必需 |
| `make_instrumental` | boolean | 可选 | 是否生成纯音乐,true 为生成纯音乐 |
| `task_id` | string | 可选 | 要续写的歌曲的任务 ID 续写时必需 |
| `continue_at` | number | 可选 | 从歌曲的第几秒开始续写 续写时必需 |
| `continue_clip_id` | string | 可选 | 要续写的歌曲的 clip ID 续写时必需 |
| `gpt_description_prompt` | string | 可选 | 灵感来源的文字描述 灵感模式必需,其他模式无需。 |
| `notify_hook` | string | 可选 | 歌曲生成完成的回调通知地址 示例 |

# 生成歌曲

根据提示生成歌曲
续写已有歌曲

## 请求参数
Header 参数生成代码Acceptstring 必需示例:application/jsonAuthorizationstring 必需示例:sk-Content-Typestring 必需示例:application/jsonBody 参数application/json必填生成代码promptstring 必需歌词内容,在自定义模式下需提供
灵感模式无需,自定义模式必需。

tagsstring 必需歌曲风格标签,使用逗号分隔,在自定义模式下需提供
灵感模式无需,自定义模式必需

mvstring 可选模型版本,可选值:chirp-v3-0、chirp-v3-5,默认为 chirp-v3-0

titlestring 必需歌曲标题,在自定义模式下需提供
灵感模式无需,自定义模式必需

make_instrumentalboolean 可选是否生成纯音乐,true 为生成纯音乐

task_idstring 可选要续写的歌曲的任务 ID
续写时必需

continue_atnumber 可选从歌曲的第几秒开始续写
续写时必需

continue_clip_idstring 可选要续写的歌曲的 clip ID
续写时必需

gpt_description_promptstring 可选灵感来源的文字描述
灵感模式必需,其他模式无需。

notify_hookstring 可选歌曲生成完成的回调通知地址

--header 'Accept: application/json' \
--header 'Authorization: sk-' \
--header 'Content-Type: application/json' \
--data-raw '{
    "prompt": "窗前明月光，疑是地上霜，举头望明月，低头思故乡",

# 查询任务状态（单个）

Suno
查询任务状态（单个）
**GET** `https://4sapi.com/suno/fetch/{{id}}`

## 请求参数

### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Authorization` | string | 必需 | sk- |  |

## 返回响应

## 请求示例

```bash
curl --location -g --request GET 'https://4sapi.com/suno/fetch/{{id}}' \
--header 'Authorization: sk-'
```

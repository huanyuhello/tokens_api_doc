# 图生视频


## 请求参数


### Header 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 示例: |
| `sk-` |  |  |  |


### Body 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `必填` |  |  |  |
| `prompt` | string |  | 视频生成的文本描述 必需 |
| `model` | string |  | 使用的模型名称 必需 |
| `images` | string | 可选 | 参考图片URL数组，用于辅助视频生成 |
| `enhance_prompt` | boolean | 可选 | 是否增强提示词，默认为 false |
| `resolution` | string |  | 分辨率 必需 |
| `aspect_ratio` | string |  | 宽高比 必需 示例 |

# 文生视频


## 请求参数


### Header 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 示例: {{Authorization}} |


### Body 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `必填` |  |  |  |
| `prompt` | string |  | 视频生成的文本描述 可选 |
| `model` | string |  | 使用的模型名称 必需 |
| `enhance_prompt` | boolean | 可选 | 是否增强提示词，默认为 false |
| `aspect_ratio` | string | 必需 | 枚举值: 16:9 横屏 9:16 竖屏 |
| `resolution` | string |  | 视频分辨率 可选 |
| `enable_upsample` | boolean | 必需 | 是否分辨率提升；返回 1080p 视频 示例 |

# 创建角色


## 请求参数


### Query 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `url` | string | 可选 |  |
| `string` | 可选 |  | 示例值: https://filesystem.site/cdn/20251030/javYrU4etHVFDqg8by7mViTWHlMOZy.mp4 |
| `timestamps` | string | 可选 |  |
| `string` | 可选 |  | 示例值: 1,3 |


### Header 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 示例: |
| `sk-` |  |  |  |


### Body 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `必填` |  |  |  |
| `url` | string | 可选 |  |
| `string` | 必需 |  | 视频 url 视频中包含需要创建的角色 |
| `timestamps` | string | 可选 |  |
| `string` | 必需 |  | 角色出现的时间范围 单位秒，例如 ‘1,2’ 是指视频的1～2秒中出现的角色，注意范围差值最大 3 秒最小 1 秒 |
| `from_task` | string | 可选 | 已完成的任务 id 可以根据已经生成的任务 id，来创建角色 示例 |

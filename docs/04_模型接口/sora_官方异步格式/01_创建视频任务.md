# 创建视频任务

官方异步格式
创建视频任务
**POST** `https://4sapi.com/v1/videos`

## 请求参数

Authorization
在 Header 添加参数 Authorization，其值为在 Bearer 之后拼接 Token
示例：
Authorization: Bearer ********************
### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Authorization` | string | 必需 | Content-Type | {{Authorization}} |
| `model` | string | 必需 |  |  |
| `prompt` | string | 必需 | seconds |  |
| `size` | string | 可选 |  |  |

## 返回响应

## 请求示例

```bash
curl --location --request POST 'https://4sapi.com/v1/videos' \
--header 'Authorization: Bearer <token>' \
--form 'model="sora-2"' \
--form 'prompt="画面动起来"' \
--form 'seconds="4"' \
--form 'input_reference=@"/Users/xiangsx/Downloads/RHTtTcBrlY5wGxop9ux8DqRckyIKxd.png"' \
--form 'size="1280x720"'
```

### Body 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `必填` |  |  |  |
| `prompt` | string | 必需 | 所需图像的文本描述。最大长度为 1000 个字符。 |
| `n` | string | 必需 | 要生成的图像数。必须介于 1 和 10 之间。 |
| `size` | string | 必需 | 生成图像的大小。必须是256x256、512x512或 1024x1024之一。 示例 |

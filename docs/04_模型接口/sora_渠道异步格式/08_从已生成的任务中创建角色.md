# 从已生成的任务中创建角色


## 请求参数


### Header 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 默认值: {{Authorization}} |


### Body 参数

| 参数名 | 类型 | 必需 | 描述 |
|--------|------|------|------|
| `character_from_task` | string | 必需 | 示例: sora-2:task_01kb8a4xs3f73vhgwjfg4zbcv0 |
| `model` | string | 必需 | 示例: |
| `sora-2-character` | chacter_timestamps |  |  |
| `string` | 可选 |  | 角色出现在视频中的时间范围，注意 end-start的差值范围在 1～3秒，格式为{start},{end} 示例: 0,3 |

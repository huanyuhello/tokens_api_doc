# API 接口返回报错，报错代码 400 401 403 404 405 413 429 500 503 504 524 是什么意思？

| 状态代码 | 代码解释 |
| --- | --- |
| 400 Bad Request | 请求格式错误或不能被服务器理解，通常意味着客户端错误，请检查你的请求格式，例如 o1 系列模型不支持 system 参数。 |
| 401 Unauthorized（无效的令牌） | API 密钥验证未通过，你需要验证你的 API 密钥是否正确，其他原因还包括令牌过期。换个模型测试来验证你的API密钥是否正确。如果你换模型正常，请联系管理员反馈处理。 |
| 403 Forbidden / 令牌分组 XXX 已被禁用 | 一般是权限不足，令牌权限问题或令牌限额用完。编辑令牌取消令牌限额 或 新建令牌可以解决。 |
| 404 Not Found | 请求的资源未找到，你可能正在尝试访问一个不存在的端点。请检查 Base Url是否填写正确，尝试带上 /v1 或最后一个斜杠 / 等。 |
| 413 Request Entity Too Large | 请求体太大，你可能需要减小你的请求体容量。Prompt可能超长了。请缩短你的prompt，再次尝试，确认短prompt是否可以正常调用。 |
| 429 Too Many Requests / 当前分组上游负载已饱和 | 由于频繁的请求超过限制，你已经超过了你的速率限制。OpenAI 对单个账号有限流，429 说明后端某个账号并发太高遇到限流，请继续尝试。 |
| 500 Internal Server Error | 服务器内部的错误，这可能是中转服务器或 OpenAI 服务器的问题，不是你的问题。请再次尝试，多次持续报错请联系管理员处理。 |
| 503 Service Unavailable / 当前分组 NNN 下对于模型 xxxx 无可用渠道 | 服务器暂时不可用，这可能是由于 OpenAI 正在进行维护或者服务器过载。也可能是中转平台后端管理的问题，请联系管理员，让他帮你加上这个模型。请再次尝试，多次持续报错请联系管理员处理。 |
| 504 Gateway Timeout | 504 错误表示“网关超时”，未能在规定时间内从上游服务器获得响应。请再次尝试，多次持续报错请联系管理员处理。 |
| 524 连接超时 | 服务器没有在规定时间内完成请求，可能是 4sapi 通道拥挤造成。请再次尝试，多次持续报错请联系管理员处理。 |
**注意**: 以上只是 HTTP 状态码的一部分，并不完全，有些状态码可能会因服务器的实现和使用而有所不同。

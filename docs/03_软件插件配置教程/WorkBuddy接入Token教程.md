# WorkBuddy 接入 Tokens 教程

> 本教程以 Windows 环境为例。

官方下载：[WorkBuddy 官网](https://workbuddy.ai)

---

## 配置步骤

### 第一步：打开模型配置

安装并打开 WorkBuddy，在主界面点击**模型** → **自定义模型**。

### 第二步：填写模型配置

以 `gpt-image-2` 生图模型为例：

| 字段 | 填写内容 |
|------|----------|
| 模型名称 | `gpt-image-2`（从 Tokens 平台模型广场复制） |
| API Base URL | `http://47.102.134.41:3000/v1` |
| API Key | `sk-...`（你的 Tokens 平台 API Key） |

填写完成后点击**保存**。

### 第三步：使用模型

在对话界面选择刚才配置的模型，输入提示词发送即可生成内容。

> **提示**：如果在模型列表中找不到刚添加的模型，退出软件重新打开即可。

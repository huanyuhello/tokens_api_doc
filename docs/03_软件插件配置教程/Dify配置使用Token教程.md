# Dify 配置使用 Tokens 教程

## Dify 简介

Dify 是一个开源的大语言模型应用开发平台，提供低代码/无代码的 AI 应用开发体验，支持从原型设计到生产部署的全流程管理。

---

## 配置步骤

### 第一步：进入模型供应商设置

登录 Dify 后，点击右上角头像 → **设置** → 左侧栏选择**模型供应商** → 点击**添加模型**。

### 第二步：配置模型

在弹出的配置窗口中填写：

| 字段 | 填写内容 |
|------|----------|
| 模型类型 | LLM |
| 模型名称 | 自定义（如 `claude-opus-4-6`） |
| API Key | 你在 Tokens 平台创建的令牌 Key |
| API endpoint URL | `http://47.102.134.41:3000` 或 `http://47.102.134.41:3000/v1` |

填写完成后点击**保存**，模型列表中会出现刚添加的模型。

### 第三步：使用模型

在 Dify 聊天对话框右上角，选择刚才添加的模型即可开始使用。

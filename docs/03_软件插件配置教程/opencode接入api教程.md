# OpenCode 接入 Tokens 教程

---

## 配置步骤

### 第一步：打开设置

启动 OpenCode 后，点击**设置**图标。

### 第二步：添加自定义提供商

1. 点击**提供商**
2. 选择**自定义提供商**，点击**连接**
3. 按如下填写配置（以 Claude 为例）：

| 字段 | 填写内容 |
|------|----------|
| 名称 | `Tokens`（自定义） |
| API Base URL | `http://47.102.134.41:3000` |
| API Key | `sk-...`（你的 Tokens 平台 API Key） |
| 模型名称 | 如 `claude-opus-4-6` |

4. 点击**提交**

### 第三步：选择模型

在模型列表中选择刚才添加的 Tokens 提供商下的模型，即可开始对话。

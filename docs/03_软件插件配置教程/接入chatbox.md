# Chatbox 接入 Tokens 教程

Chatbox 是一款跨平台的 AI 桌面客户端，支持多种 API 接入。

官方下载：[https://chatboxai.app](https://chatboxai.app)

---

## 配置步骤

### 第一步：打开设置

安装并打开 Chatbox 后，点击左下角**设置**图标。

### 第二步：添加 AI 服务商

点击**添加**，填写以下信息：

| 字段 | 填写内容 |
|------|----------|
| 名称 | `Tokens`（自定义） |
| API 模式 | `Claude API 兼容` 或 `OpenAI API 兼容` |
| API Host | `http://47.102.134.41:3000` |
| API Key | `sk-...`（你的 Tokens 平台 API Key） |

### 第三步：选择模型

在模型列表中输入或选择模型名称（如 `claude-opus-4-6`），点击保存。

### 第四步：开始使用

在对话界面选择刚才配置的服务商和模型，即可开始使用。

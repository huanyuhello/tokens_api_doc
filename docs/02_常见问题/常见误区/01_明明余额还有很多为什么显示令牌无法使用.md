# 明明余额还有很多，为什么显示令牌无法使用？

!!! warning "注意区分两种余额"
    **账户余额** 和 **令牌额度** 是两个独立的概念，账户有钱不代表令牌可以无限使用。

## 常见原因排查

### 原因一：令牌设置了额度限制

令牌有自己独立的额度上限。当令牌额度用尽时，即使账户余额充足也无法继续调用。

**解决方法：** 进入 **「密钥管理」**，编辑对应令牌，取消额度限制或调高额度上限。

### 原因二：令牌开启了 IP 白名单

令牌启用了白名单功能，限制了可访问的 IP 地址，当前请求的 IP 不在白名单内。

**解决方法：** 编辑令牌，关闭 IP 白名单，或将当前 IP 添加至白名单。

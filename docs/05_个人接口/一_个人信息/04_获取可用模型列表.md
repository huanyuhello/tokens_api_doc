# 获取可用模型列表

**GET** `https://tokens.smartfashionai.cn/api/user/models`

返回当前用户可调用的所有模型 ID 列表。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data` | array[string] | 可用模型 ID 列表 |
| `message` | string | 返回信息 |
| `success` | boolean | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/user/models' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

---

## 响应示例

```json
{
    "data": [
        "gemini-2.5-pro",
        "gemini-2.5-flash",
        "claude-sonnet-4-6",
        "claude-opus-4-6",
        "gpt-5",
        "gpt-4o",
        "o3",
        "..."
    ],
    "message": "",
    "success": true
}
```

> 实际返回的模型列表以控制台「模型广场」为准，上方仅为示例节选。

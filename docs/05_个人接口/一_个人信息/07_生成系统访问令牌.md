# 生成系统访问令牌

**GET** `https://tokens.smartfashionai.cn/api/user/token`

生成用于调用个人接口的系统访问令牌。生成的 Token 需放入后续请求的 `Authorization` Header 中使用。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 已有的系统访问令牌（首次生成时可留空） |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data` | string | 生成的系统访问令牌 |
| `message` | string | 返回信息 |
| `success` | boolean | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/user/token' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

---

## 响应示例

```json
{
    "data": "生成的token",
    "message": "",
    "success": true
}
```

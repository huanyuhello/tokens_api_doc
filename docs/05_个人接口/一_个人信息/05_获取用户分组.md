# 获取用户分组

**GET** `https://tokens.smartfashionai.cn/api/user/self/groups`

返回当前用户可用的所有分组及其描述和倍率信息。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data` | object | 分组信息，key 为分组名称 |
| `data.<分组名>.desc` | string | 分组描述 |
| `data.<分组名>.ratio` | number | 计费倍率 |
| `message` | string | 返回信息 |
| `success` | boolean | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/user/self/groups' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

# 注销账户

**DELETE** `https://tokens.smartfashionai.cn/api/user/self`

> 注意：此操作不可逆，账户注销后数据将无法恢复。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

---

## 请求示例

```bash
curl --location --request DELETE 'https://tokens.smartfashionai.cn/api/user/self' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

---

## 响应示例

```json
{}
```

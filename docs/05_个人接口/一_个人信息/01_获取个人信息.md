# 获取个人信息

**GET** `https://tokens.smartfashionai.cn/api/user/self`

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

---

## 返回字段

| 字段 | 说明 |
|------|------|
| `data.id` | 用户 ID |
| `data.username` | 用户名 |
| `data.display_name` | 显示名称 |
| `data.email` | 邮箱 |
| `data.group` | 所属分组 |
| `data.quota` | 剩余额度 |
| `data.used_quota` | 已用额度 |
| `data.request_count` | 请求次数 |
| `data.role` | 角色（1=普通用户） |
| `data.status` | 状态（1=正常） |
| `data.aff_code` | 邀请码 |
| `data.permissions` | 权限配置 |
| `message` | 返回信息 |
| `success` | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/user/self' \
--header 'Authorization: token' \
--header 'New-Api-User: 126638'
```

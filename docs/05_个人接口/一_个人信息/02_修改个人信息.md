# 修改个人信息

**PUT** `https://tokens.smartfashionai.cn/api/user/self`

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

### Body（application/json）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `username` | string | 可选 | 用户名（≤20 字符） |
| `display_name` | string | 可选 | 显示名称（≤20 字符） |
| `original_password` | string | 可选 | 原密码 |
| `password` | string | 可选 | 新密码（8-20 字符） |
| `sidebar_modules` | string | 可选 | 更新侧边栏模块 |
| `language` | string | 可选 | 更新语言偏好 |

---

## 请求示例

```bash
curl --location --request PUT 'https://tokens.smartfashionai.cn/api/user/self' \
--header 'Authorization: token' \
--header 'New-Api-User: 126638' \
--header 'Content-Type: application/json' \
--data-raw '{
    "username": "name",
    "display_name": "name",
    "original_password": "old pass",
    "password": "new pass"
}'
```

---

{}

# 令牌模型消耗预览（Top 3）

**GET** `https://tokens.smartfashionai.cn/api/log/self/token_daily/model_preview`

获取指定令牌在时间范围内消耗最多的前 3 个模型及其每日数据。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `token_id` | integer | 可选 | 令牌 ID |
| `start_timestamp` | integer | 可选 | 开始时间戳（秒） |
| `end_timestamp` | integer | 可选 | 结束时间戳（秒） |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.top_models` | array | Top 3 模型列表 |
| `data.top_models[].model_name` | string | 模型名称 |
| `data.top_models[].total_cost` | integer | 总消耗额度 |
| `data.top_models[].total_count` | integer | 总请求次数 |
| `data.top_models[].daily_data` | object | 每日消耗（key 为日期时间戳） |
| `data.other_models_summary.count` | integer | 其他模型请求次数 |
| `data.other_models_summary.total_cost` | integer | 其他模型总消耗 |
| `data.has_more` | boolean | 是否有更多模型 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/log/self/token_daily/model_preview?token_id=7652&start_timestamp=1773974400&end_timestamp=1774032510' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

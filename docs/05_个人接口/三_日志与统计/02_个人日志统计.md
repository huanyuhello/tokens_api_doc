# 个人日志统计

**GET** `https://tokens.smartfashionai.cn/api/log/self/stat`

获取当前用户在指定时间范围内的消耗统计汇总。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `type` | string | 可选 | 日志类型 |
| `start_timestamp` | string | 可选 | 开始时间戳 |
| `end_timestamp` | string | 可选 | 结束时间戳 |
| `token_name` | string | 可选 | 令牌名称 |
| `model_name` | string | 可选 | 模型名称 |
| `channel` | integer | 可选 | 渠道 ID |
| `group` | string | 可选 | 分组 |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.quota` | integer | 总消耗额度 |
| `data.rpm` | integer | 每分钟请求数 |
| `data.tpm` | integer | 每分钟 Token 数 |
| `message` | string | 返回信息 |
| `success` | boolean | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/log/self/stat?type=&start_timestamp=&end_timestamp=&token_name=&model_name=&channel=&group=' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

---

## 响应示例

```json
{
    "data": {
        "quota": 32983187,
        "rpm": 0,
        "tpm": 0
    },
    "message": "",
    "success": true
}
```

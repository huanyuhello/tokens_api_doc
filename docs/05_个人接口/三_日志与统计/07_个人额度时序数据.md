# 个人额度时序数据

**GET** `https://tokens.smartfashionai.cn/api/data/self`

获取当前用户在指定时间范围内的额度消耗时序数据（时间跨度不超过 30 天）。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `start_timestamp` | integer | 必需 | 开始时间戳（秒） |
| `end_timestamp` | integer | 必需 | 结束时间戳（秒，与开始时间跨度 ≤30 天） |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data` | array | 时序数据列表 |
| `data[].id` | integer | 记录 ID |
| `data[].user_id` | integer | 用户 ID |
| `data[].model_name` | string | 模型名称 |
| `data[].created_at` | integer | 时间戳（秒） |
| `data[].token_used` | integer | Token 用量 |
| `data[].count` | integer | 请求次数 |
| `data[].quota` | integer | 消耗额度 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/data/self?start_timestamp=1773974400&end_timestamp=1774032510' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

# 令牌全量模型消耗

**GET** `https://tokens.smartfashionai.cn/api/log/self/token_daily/model_full`

获取指定令牌在时间范围内所有模型的消耗汇总。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `token_id` | integer | 可选 | 令牌 ID |
| `start_timestamp` | integer | 可选 | 开始时间戳（秒） |
| `end_timestamp` | integer | 可选 | 结束时间戳（秒） |
| `sort_by` | string | 可选 | 排序字段 |
| `sort_order` | string | 可选 | 排序方向：`asc` / `desc` |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data` | array | 模型消耗列表 |
| `data[].model_name` | string | 模型名称 |
| `data[].total_cost` | integer | 总消耗额度 |
| `data[].total_count` | integer | 总请求次数 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/log/self/token_daily/model_full?token_id=7652&start_timestamp=1773974400&end_timestamp=1774032510&sort_by=&sort_order=' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

---

## 响应示例

```json
{
    "data": [
        {
            "model_name": "veo3.1",
            "total_cost": 1000000,
            "total_count": 2
        }
    ],

    "success": true

```

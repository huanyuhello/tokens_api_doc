# 删除令牌

**DELETE** `https://tokens.smartfashionai.cn/api/token/{id}`

根据令牌 ID 删除单个令牌。

---

## 请求参数

### Path

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `id` | integer | 必需 | 令牌 ID |

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

---

## 请求示例

```bash
curl --location --request DELETE 'https://tokens.smartfashionai.cn/api/token/7727' \
--header 'Authorization: user-token' \
--header 'New-Api-User: 125259'
```

---

## 响应示例

```json
{
    "message": "",
    "success": true
}
```

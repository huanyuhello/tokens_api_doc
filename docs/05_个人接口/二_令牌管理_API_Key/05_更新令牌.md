# 更新令牌

**PUT** `https://tokens.smartfashionai.cn/api/token/`

更新令牌信息。添加 `?status_only` 查询参数时，仅更新令牌状态字段。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `status_only` | - | 可选 | 存在时仅更新 `status` 字段 |

### Body（application/json）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `id` | integer | 必需 | 令牌 ID |
| `status` | integer | 可选 | 状态值（1=启用，2=禁用） |

---

## 请求示例

```bash
curl --location --request PUT 'https://tokens.smartfashionai.cn/api/token/?status_only' \
--header 'Authorization: {access_token}' \
--header 'New-Api-User: {user_id}' \
--header 'Content-Type: application/json' \
--data-raw '{
    "id": 7727,
    "status": 1
}'
```

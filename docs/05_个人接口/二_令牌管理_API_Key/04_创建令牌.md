# 创建令牌

**POST** `https://tokens.smartfashionai.cn/api/token/`

创建一个新的 API Key。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

### Body（application/json）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `name` | string | 必需 | 令牌名称（≤50 字符） |
| `expired_time` | integer | 可选 | 过期时间戳（`-1` = 永不过期） |
| `remain_quota` | integer | 可选 | 初始额度（非无限时有效，范围 0 ~ 10亿×QuotaPerUnit） |
| `unlimited_quota` | boolean | 可选 | 是否无限额度 |
| `model_limits_enabled` | boolean | 可选 | 是否启用模型限制 |
| `model_limits` | string | 可选 | 可用模型列表（逗号分隔） |
| `allow_ips` | string | 可选 | IP 白名单（换行分隔） |
| `group` | string | 可选 | 所属分组 |
| `cross_group_retry` | boolean | 可选 | 是否启用跨分组重试 |

---

## 请求示例

```bash
curl --location --request POST 'https://tokens.smartfashionai.cn/api/token/' \
--header 'Authorization: {access_token}' \
--header 'New-Api-User: {user_id}' \
--header 'Content-Type: application/json' \
--data-raw '{
    "name": "Tokens",
    "expired_time": 1769280706264,
    "remain_quota": 18980,
    "unlimited_quota": false,
    "model_limits_enabled": true,
    "model_limits": "gpt-5.4",
    "group": "Gpt-codex",
    "cross_group_retry": false
}'
```

---

## 响应示例

```json
{
    "message": "",
    "success": true
}
```

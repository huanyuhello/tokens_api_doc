# 批量删除令牌

**POST** `https://tokens.smartfashionai.cn/api/token/batch`

批量删除多个令牌。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

### Body（application/json）

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `ids` | array[integer] | 必需 | 要删除的令牌 ID 数组 |

---

## 请求示例

```bash
curl --location --request POST 'https://tokens.smartfashionai.cn/api/token/batch' \
--header 'Authorization: user-token' \
--header 'New-Api-User: 125259' \
--header 'Content-Type: application/json' \
--data-raw '{
    "ids": [7132]
}'
```

---

## 响应示例

```json
{
    "data": 1,
    "message": "",
    "success": true
}
```

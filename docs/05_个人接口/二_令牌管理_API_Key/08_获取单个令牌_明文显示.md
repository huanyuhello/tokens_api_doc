# 获取单个令牌（明文显示）

**POST** `https://tokens.smartfashionai.cn/api/token/{id}/key`

获取令牌的完整明文 Key（`sk-` 开头的完整字符串）。

---

## 请求参数

### Path

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `id` | integer | 必需 | 令牌 ID |

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌（Bearer 格式） |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

---

## 请求示例

```bash
curl --location --request POST 'https://tokens.smartfashionai.cn/api/token/6820/key' \
--header 'Authorization: Bearer user-token' \
--header 'New-Api-User: user-id'
```

---

## 响应示例

```json
{
    "data": {
        "key": "sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
    },
    "message": "",
    "success": true
}
```

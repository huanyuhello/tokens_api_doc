# 搜索令牌

**GET** `https://tokens.smartfashionai.cn/api/token/search`

按名称或 Key 搜索令牌。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `keyword` | string | 可选 | 按名称搜索，支持 `%` 模糊匹配，去掉 `%` 后关键词需 ≥2 字符 |
| `token` | string | 可选 | 按 Key 搜索（自动去掉 `sk-` 前缀） |
| `page` | integer | 可选 | 页码 |
| `page_size` | integer | 可选 | 每页条数 |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.page` | integer | 当前页码 |
| `data.page_size` | integer | 每页条数 |
| `data.total` | integer | 总条数 |
| `data.items` | array | 令牌列表 |
| `message` | string | 返回信息 |
| `success` | boolean | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/token/search?keyword=&token=&page=&page_size=1' \
--header 'Authorization: token' \
--header 'New-Api-User: id'
```

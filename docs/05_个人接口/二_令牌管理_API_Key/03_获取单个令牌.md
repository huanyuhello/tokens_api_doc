# 获取单个令牌

**GET** `https://tokens.smartfashionai.cn/api/token/{id}`

根据令牌 ID 获取单个令牌的详细信息。

---

## 请求参数

### Path

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `id` | integer | 必需 | 令牌 ID |

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 可选 | 系统访问令牌 |
| `New-Api-User` | string | 可选 | 用户 ID（数字） |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/token/6820' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

# 获取令牌列表

**GET** `https://tokens.smartfashionai.cn/api/token/`

分页获取当前用户的所有 API Key 列表。

---

## 请求参数

### Header

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `Authorization` | string | 必需 | 系统访问令牌 |
| `New-Api-User` | string | 必需 | 用户 ID（数字） |

### Query

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `page` | integer | 可选 | 页码，默认 1 |
| `page_size` | integer | 可选 | 每页条数 |

---

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `data.page` | integer | 当前页码 |
| `data.page_size` | integer | 每页条数 |
| `data.total` | integer | 总条数 |
| `data.items` | array | 令牌列表 |
| `data.items[].id` | integer | 令牌 ID |
| `data.items[].name` | string | 令牌名称 |
| `data.items[].key` | string | API Key |
| `data.items[].status` | integer | 状态（1=启用） |
| `data.items[].group` | string | 所属分组 |
| `data.items[].remain_quota` | integer | 剩余额度 |
| `data.items[].unlimited_quota` | boolean | 是否无限额度 |
| `data.items[].used_quota` | integer | 已用额度 |
| `data.items[].expired_time` | integer | 过期时间戳（-1=永不过期） |
| `message` | string | 返回信息 |
| `success` | boolean | 是否成功 |

---

## 请求示例

```bash
curl --location --request GET 'https://tokens.smartfashionai.cn/api/token/' \
--header 'Authorization: token' \
--header 'New-Api-User: 126638'
```

# 价格-充值-分组-模型价格介绍

一、关于分组价格与倍率的解释

计费原则：全部模型同步各大模型输入输出计费逻辑，我们Tokens的计费是在官方的基础上做折扣。

网站的在线充值比例：1人民币=1美金。

倍率扣费逻辑：1倍率的话1美金=扣1人民币，2倍率1美金=扣2人民币，6倍率1美金的=扣6人民币。

我们说的3折就是7✖️0.3约等于2.1元，8.5折就是7✖️0.85约等于6元。

所以：1倍率意思就是:1元/刀。冲1块钱就可以当官方的1美金使用。

价格说明：价格1.5，就是1.5人民币=1美金，在分组中显示为1.5倍率，以此为例，倍率越高，价格越高。

定价逻辑：4-7块=1美金的都是官方的KEY，放在我们Tokens中转站转发，比较昂贵，需要考虑自身的需求做决定，例如claudecode企业分组，用MAX账号的号池API，功能智商=官方，性价比优选，但部分项目只能用官KEY除外，总结价格低稳定性和适配肯定会不如官KEY，但是也就是98%和99%这样的区别，阶段性官方风控的话，稳定性还会略微降低。

一key通：就是一个API通过国内外各大模型，无需换KEY直接切换模型即可，价格1.5的属于全模型特价渠道，价格为7的属于全模型官KEY中转。

二、关于充值

如下图所示：
1、站点支持自助充值，点击【钱包管理】

2、点击【账单】，可以查看充值记录

3、点击【自定义】，可以充值自己想要的金额，如遇到白屏，则关掉梯子或者切换网络重试！

4、关于开票，联系客服，提供【用户名】+【开票信息】+【开票金额】即可开票，由于财务流程繁琐，低于1000元我司作为无票收入入账，高于1000元的可联系客服申请开票，金额可以累计。

三、分组的讲解

分组价格：分组不同，渠道不同，价格不同，智力跟稳定性、性能都不同，越贵越优质。

选择分组：高速、稳定、便宜是不可能三角。选择适合自己产品的分组即可，请不要奢望低折扣拿到=官方的API甚至稳定性高于官方的API。

分组优先级：平台会先把任务打到第一个分组，失败重试到第二个分组，从而实现整体成功率。

四、关于模型价格与配置

1、具体模型价格需要在【模型广场中查找】；

2、搜索框输入模型，支持模糊搜索；

3、点击想要的模型，可以看到模型输入输出价格；

4、点击倍率展示，点击CNY，展示人民币报价；

5、模型配置时名字必须一模一样，建议在模型广场复制模型名称调用。

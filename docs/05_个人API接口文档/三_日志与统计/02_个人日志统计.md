# 个人日志统计

三、日志与统计
2. 个人日志统计
**GET** `http://47.102.134.41:3000/api/log/self/stat`

## 请求参数

### Query 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `type` | string |  | 日志类型 |  |
| `start_timestamp` | string |  | 开始时间戳 |  |
| `end_timestamp` | string |  | 结束时间戳 |  |
| `token_name` | string |  | 令牌名称 |  |
| `model_name` | string |  | 模型名称 |  |
| `channel` | integer |  | 渠道 ID |  |
| `group` | string |  | 分组 |  |

### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Authorization` | string | 必需 | New-Api-User |  |

## 返回响应

data
必需
quota
必需
rpm
必需
tpm
必需
message
必需
success
必需
## 请求示例

```bash
curl --location --request GET 'http://47.102.134.41:3000/api/log/self/stat?type=&start_timestamp=&end_timestamp=&token_name=&model_name=&channel=&group=' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

## 响应示例

```json
{"data":{"quota":32983187,"rpm":0,"tpm":0},"message":"","success":true}
```

# 1. 查看个人使用日志

日志类型（0=未知, 1=充值, 2=消费, 3=管理, 4=系统）

start_timestampinteger 开始时间戳（秒）可选end_timestampinteger 结束时间戳（秒）可选token_namestring 按令牌名称过滤可选model_namestring 按模型名称过滤可选groupstring 按分组过滤可选request_idstring 按请求 ID 过滤可选Header 参数生成代码Authorizationstring 必需示例:user-tokenNew-Api-Userstring 必需示例:user-id
## 返回响应
🟢200成功application/json生成代码Body生成代码dataobject 必需pageinteger 必需page_sizeinteger 必需totalinteger 必需itemsarray [object] 必需messagestring 必需successboolean 必需请求示例请求示例ShellJavaScriptJavacURLcURL-WindowsHttpiewgetPowerShell
```
curl --location --request GET 'https://4sapi.com/api/log/self?page=1&page_size=&type=&start_timestamp=&end_timestamp=&token_name=&model_name=&group=&request_id=' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

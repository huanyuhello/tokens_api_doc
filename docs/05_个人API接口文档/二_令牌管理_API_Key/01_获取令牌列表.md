# 获取令牌列表

二、令牌管理（API Key）
1. 获取令牌列表
**GET** `http://47.102.134.41:3000/api/token/`

## 请求参数

### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Authorization` | string | 必需 | New-Api-User |  |

## 返回响应

data
返回数据
必需
page
必需
page_size
必需
total
必需
items
array [object]
必需
message
返回信息
必需
success
是否成功
必需
## 请求示例

```bash
curl --location --request GET 'http://47.102.134.41:3000/api/token/' \
--header 'Authorization: token' \
--header 'New-Api-User: 126638'
```

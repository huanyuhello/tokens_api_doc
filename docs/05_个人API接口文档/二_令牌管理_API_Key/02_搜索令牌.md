# 2.搜索令牌

按名称搜索（支持 `%` 模糊，去掉 `%` 后关键词≥2字符）

tokenstring 可选按 key 搜索（自动去掉 `sk-` 前缀

pageinteger 页码可选page_sizeinteger 每页条数可选示例:1Header 参数生成代码Authorizationstring 可选系统访问令牌（个人设置中生成）

示例:tokenNew-Api-Userstring 用户 ID（数字）可选示例:id
## 返回响应
🟢200成功application/json生成代码Body生成代码dataobject 必需pageinteger 必需page_sizeinteger 必需totalinteger 必需itemsarray [object] 必需messagestring 必需successboolean 必需请求示例请求示例ShellJavaScriptJavacURLcURL-WindowsHttpiewgetPowerShell
```
curl --location --request GET 'https://tokens.smartfashionai.cn//api/token/search?keyword=&token=&page=&page_size=1' \
--header 'Authorization: token' \
--header 'New-Api-User: id'
```

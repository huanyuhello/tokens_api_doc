# 获取单个令牌 明文显示

二、令牌管理（API Key）
8.获取单个令牌（明文显示）
**POST** `http://47.102.134.41:3000/api/token/令牌ID/key`

## 请求参数

### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Authorization` | string | 可选 | New-Api-User |  |

## 返回响应

## 请求示例

```bash
curl --location --request POST 'http://47.102.134.41:3000/api/token/令牌ID/key' \
--header 'Authorization: Bearer y3hbrV8***************' \
--header 'New-Api-User: 126***********'
```

## 响应示例

```json
{
    "data": {
        "key": "YuMwaJo9K。。。。。。。。。。"
    },
    "message": "",
    "success": true
}
```

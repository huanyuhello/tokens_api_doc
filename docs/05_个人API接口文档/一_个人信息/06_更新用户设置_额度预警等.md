# 6. 更新用户设置（额度预警等）

预警类型：email / webhook / bark / gotify

quota_warning_thresholdinteger 必需当钱包或订阅剩余额度低于此数值时，系统将通过选择的方式发送通知

notification_emailstring 可选通知邮箱（email 类型时）email

webhook_urlstring 可选Webhook 地址

webhook_secretstring 可选Webhook 密钥

bark_urlstring Bark 推送地址可选gotify_urlstring 可选Gotify 服务器地址

gotify_tokenstring Gotify 令牌可选gotify_priorityinteger 可选Gotify 优先级

accept_unset_model_ratio_modelboolean 可选是否接受未设置倍率的模型

record_ip_logboolean 可选是否记录 IP 日志

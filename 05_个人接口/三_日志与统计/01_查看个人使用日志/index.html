    查看个人使用日志¶

    GET https://tokens.smartfashionai.cn/api/log/self

    分页查询当前用户的使用日志。

    请求参数¶

    Header¶

    参数名 类型 必填 说明
    Authorization string 必需 系统访问令牌
    New-Api-User string 必需 用户 ID（数字）

    Query¶

    参数名 类型 必填 说明
    page integer 可选 页码，默认 1
    page_size integer 可选 每页条数
    type integer 可选 日志类型（0=未知，1=充值，2=消费，3=管理，4=系统）
    start_timestamp integer 可选 开始时间戳（秒）
    end_timestamp integer 可选 结束时间戳（秒）
    token_name string 可选 按令牌名称过滤
    model_name string 可选 按模型名称过滤
    group string 可选 按分组过滤
    request_id string 可选 按请求 ID 过滤

    返回字段¶

    字段 类型 说明
    data.page integer 当前页码
    data.page_size integer 每页条数
    data.total integer 总条数
    data.items array 日志列表
    data.items[].id integer 日志 ID
    data.items[].type integer 日志类型
    data.items[].content string 日志内容
    data.items[].model_name string 模型名称
    data.items[].quota integer 消耗额度
    data.items[].prompt_tokens integer 输入 Token 数
    data.items[].completion_tokens integer 输出 Token 数
    data.items[].created_at integer 创建时间戳
    message string 返回信息
    success boolean 是否成功

    请求示例¶

    curl --location --request GET 'https://tokens.smartfashionai.cn/api/log/self?page=1&page_size=10&type=2' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'

# 从已生成的任务中创建角色

默认值:
{{Authorization}}

Body 参数application/x-www-form-urlencoded必填生成代码character_from_taskstring 必需示例:sora-2:task_01kb8a4xs3f73vhgwjfg4zbcv0modelstring 必需示例:sora-2-characterchacter_timestampsstring 可选角色出现在视频中的时间范围，注意 end-start的差值范围在 1～3秒，格式为{start},{end}

示例:0,3
## 返回响应
🟢200成功application/json生成代码Body生成代码idstring 必需usernamestring 必需permalinkstring 必需profile_picture_urlstring 必需请求示例请求示例ShellJavaScriptJavacURLcURL-WindowsHttpiewgetPowerShell
```
curl --location --request POST 'https://4sapi.com/v1/videos' \
--header 'Authorization: Bearer ' \
--data-urlencode 'character_from_task=sora-2:task_01kb8a4xs3f73vhgwjfg4zbcv0' \
--data-urlencode 'model=sora-2-character' \
--data-urlencode 'chacter_timestamps=0,3'
```

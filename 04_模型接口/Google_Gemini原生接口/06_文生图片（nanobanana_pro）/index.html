    文生图片（nanobanana pro）¶

    Google Gemini接口 文生图片（nanobanana pro） POST https://4sapi.com/v1beta/models/gemini-3-pro-image-preview:generateContent

    请求参数¶

    Authorization 在 Header 添加参数 Authorization，其值为在 Bearer 之后拼接 Token 示例： Authorization: Bearer ****

    Header 参数¶

    参数名 类型 必填 说明 示例
    Authorization string 必需 Content-Type

    Body 参数¶

    参数名 类型 必填 说明 示例
    role string 可选 array [object]
    generationConfig object 必需 array[string]

    返回响应¶

    candidates array [object] 必需 content 可选 finishReason 可选 index 可选 usageMetadata 必需 promptTokenCount 必需 candidatesTokenCount 必需 totalTokenCount 必需 thoughtsTokenCount 必需 promptTokensDetails array [object] 必需 candidatesTokensDetails array [object] 必需 modelVersion 必需 responseId 必需

    请求示例¶

    curl --location --request POST 'https://4sapi.com/v1beta/models/gemini-3-pro-image-preview:generateContent' \
--header 'Authorization: Bearer <token>' \
--header 'Content-Type: application/json' \
--data-raw '{
  "contents": [
    {
      "role": "user",
      "parts": [
        {
          "text": "生成一张香水宣传海报"
        }
      ]
    }
  ],
  "generationConfig": {
    "responseModalities": [
      "TEXT",
      "IMAGE"
    ]
  }
}'

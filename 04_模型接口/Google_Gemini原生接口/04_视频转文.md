# 视频转文

Google Gemini接口
视频转文
**POST** `https://4sapi.com/v1beta/models/gemini-2.5-flash:generateContent`

## 请求参数

### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Accept` | string | 必需 | Authorization |  |
| `Content-Type` | string | 必需 | application/json |  |

### Body 参数

## 返回响应

## 请求示例

```bash
curl --location --request POST 'https://4sapi.com/v1beta/models/gemini-2.5-flash:generateContent' \
--header 'Accept: application/json' \
--header 'Authorization: sk-' \
--header 'Content-Type: application/json' \
--data-raw '{
    "contents": [
        {
            "role": "user",
            "parts": [
                {
                    "text": "关于图片，写一个关于魔法背包的故事"
                },
                {
                    "inline_data": {
                        "mime_type": "video/mp4",
                        "data": "AA。。。。。。"
                    }
                }
            ]
        }
    ]
}'
```

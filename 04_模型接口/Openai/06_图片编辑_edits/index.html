    请求参数¶

    Header 参数¶

    参数名 类型 必需 描述
    Accept string 必需 示例: application/json
    Authorization string 必需 示例:
    sk-
    Content-Type string 必需 示例: application/json

    Body 参数 (multipart/form-data)¶

    参数名 类型 必需 描述
    model string 可选 image[]
    file 可选
    prompt string 可选
    quality string 可选

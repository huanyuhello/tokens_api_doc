    批量创建嵌入¶

    请求参数¶

    Header 参数¶

    参数名 类型 必需 描述
    Authorization string 必需 示例:
    sk-

    Body 参数¶

    参数名 类型 必需 描述
    input array[string] 必需 输入文本以获取嵌入，编码为字符串或标记数组。要在单个请求中获取多个输入的嵌入，请传递一个字符串数组或令牌数组数组。每个输入的长度不得超过 8192 个标记。
    model string 必需 要使用的模型的 ID。您可以使用List models API 来查看所有可用模型，或查看我们的模型概述以了解它们的描述。
    encoding_format string 必需 示例

# 生成系统访问令牌

一、个人信息
7. 生成系统访问令牌
**GET** `https://4sapi.com/api/user/token`

## 请求参数

### Header 参数

| 参数名 | 类型 | 必填 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| `Authorization` | string | 可选 | New-Api-User |  |

## 返回响应

## 请求示例

```bash
curl --location --request GET 'https://4sapi.com/api/user/token' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
```

## 响应示例

```json
{
    "data": "生成的token",
    "message": "",
    "success": true
}
```

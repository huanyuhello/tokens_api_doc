    获取可用模型列表¶

    一、个人信息 4. 获取可用模型列表 GET http://47.102.134.41:3000/api/user/models

    请求参数¶

    Header 参数¶

    参数名 类型 必填 说明 示例
    Authorization string 必需 New-Api-User

    返回响应¶

    data array[string] 必需 message 必需 success 必需

    请求示例¶

    curl --location --request GET 'http://47.102.134.41:3000/api/user/models' \
--header 'Authorization: user-token' \
--header 'New-Api-User: user-id'
